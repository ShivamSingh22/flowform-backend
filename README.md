# FlowForm

A JSON-driven dynamic form builder with real-time collaboration presence.

## What it does
- Build forms visually by dragging and dropping fields
- Generates a JSON config behind the scenes
- Renders that JSON as a real, submittable form
- Shows live presence — who is viewing the form right now

## Tech Stack
- Backend: NestJS, PostgreSQL, Prisma, JWT, Websockets
- Frontend: React, TypeScript, Zustand, Vite
